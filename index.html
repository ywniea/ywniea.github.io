Event loop in js

2018-11-17

Event loop

JS是单线程的。只有一个调用栈call stack，每次只能做一件事情。主线程会首先去执行同步代码。遇到异步代码就会挂起，然后继续执行后面的同步代码。
当异步代码返回结果，会把这个结果放到task队列中。当主线程的执行栈中没有任务了，就回去task队列中取出异步任务的结果来执行。

React memo

2018-01-09

React design

在某一时间节点调用 React 的 render() 方法，会创建一棵由 React 元素组成的树。在下一次 state 或 props 更新时，相同的 render() 方法会返回一棵不同的树。React 需要基于这两棵树之间的差别来判断如何有效率的更新 UI 以保证当前 UI 与最新的树保持同步。
